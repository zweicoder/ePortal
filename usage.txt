Usage: eportal <username> <password> <path> // if installed with -g

Otherwise cd path/toRepo and run node cli.js <username> <password> <path>

Required options

username           your username
password           your path
path               path to the file relative to current directory (Optional, defaults to folder specified in settings.js)

Example            cd path/to/output/files && eportal myusername mypassword . // install everything into path/to/output/files
